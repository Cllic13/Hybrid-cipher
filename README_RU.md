# CRYPT-OS: Гибридный Шифр На Основе Шифра Виженера
> **Инструмент симметричного потокового шифрования с визуальным терминалом**

[English version 🇬🇧](Readme_EN.md) | **Русская версия 🇷🇺**


## О проекте
**CRYPT-OS** — это авторская реализация симметричного алгоритма шифрования, которая объединяет классическую замену символов и динамическое вычисление разности индексов. Полная версия программы воссоздает атмосферу хакерского терминала из кино, сопровождая каждый этап работы (генерация ключа, анализ данных, шифрование) анимациями и логами в реальном времени.

## Ключевые особенности
*   **Гибридная логика:** Алгоритм не просто использует шифр Виженера или Цезаря, а совмещает их (хоть и очень просто).
*   **Простота понимания:** Сам алгоритм не представляет особой сложности, он просто работает.
*   **Случайность ключа:** Ключ не имеет фиксированного размера, а также число цифр в нём случайно.
*   **Простенький шифр ключа:** Ключ защищён лёгким шифром, что добавит сложности к его использованию без алгоритма.
*   **Визуальный интерфейс:** У полной версии (HollyWood Edition) "хакерский" интерфейс.
*   **Поддержка пробелов:** Полная поддержка латиницы и пробелов. Поддержка кириллицы, спец символов, цифр и т.п. только планируется.
## Технический стек
*   **Язык:** Python 3.XX
*   **Библиотеки:** `random`, `time`, `sys`
